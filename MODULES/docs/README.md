This directory contains narrative Canon pages — design documents, runbooks,
Canon references, and ritual specifications.

## Document Types

| Type | Purpose |
|---|---|
| Design doc | Explains the rationale and architecture of a system component |
| Runbook | Step-by-step operational guide |
| Canon reference | Authoritative statement of Canon 6.0 rules |
| Ritual specification | Formal specification of a Quantumonix ceremony |

## Naming Convention

`DOC-<type>-<number>-<short-name>.md`

Example: `DOC-design-001-origin-anchoring.md`

## Required Metadata

Every doc is a canonical artifact and must include the canonical header
(see `CANON/canon_6.0_reference.md`).
