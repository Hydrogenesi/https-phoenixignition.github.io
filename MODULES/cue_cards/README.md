This directory contains cue cards — small, atomic reference modules used for
onboarding, quick lookup, and ceremony prompts.

## Types of Cue Cards

- **Law cue cards** — Summary of a TUA law
- **Operator cue cards** — Summary of a TUA operator
- **Ceremony cue cards** — Step-by-step ceremony prompt
- **Pattern cue cards** — Reusable transformation patterns

## Naming Convention

`CUE-<type>-<number>-<short-name>.md`

Example: `CUE-law-001-tension-precedes-action.md`

## Required Metadata

Every cue card must include the canonical header (see `CANON/canon_6.0_reference.md`).
