This directory contains TUA operator definitions. Operators are the agents of
transformation that act on artifacts under the governance of TUA laws and
Canon 6.0 scaling constraints.

## Naming Convention

`TUA-OP-<number>-<short-name>.md`

Example: `TUA-OP-001-tension-resolver.md`

## Required Metadata

Every operator artifact must include the canonical header (see `CANON/canon_6.0_reference.md`).
