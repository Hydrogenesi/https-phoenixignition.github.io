This directory contains TUA law artifacts. Each law is an atomic, named rule that
governs operator behavior and tension resolution within the CVN·A_TUA_M system.

## Naming Convention

`TUA-LAW-<number>-<short-name>.md`

Example: `TUA-LAW-001-tension-must-precede-action.md`

## Required Metadata

Every law artifact must include the canonical header (see `CANON/canon_6.0_reference.md`).
