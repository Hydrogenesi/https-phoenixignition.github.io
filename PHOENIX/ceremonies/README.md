This directory contains ceremony records generated by the Phoenix Quantumonix
ceremony runner.

## Naming Convention

`PHX-QX-YYYY-MM-DD-CVN.md`

Example: `PHX-QX-2026-03-03-CVN.md`

## Ceremony Structure

Each ceremony record documents the four phases:

1. **Call** — Ceremony ID, initiator, tension reference
2. **Witness** — Roles: initiator, witness, steward
3. **Transform** — Operators applied, artifacts changed
4. **Seal** — Badges issued, lineage updated, logs written

## Required Metadata

Every ceremony record must include the canonical header (see `CANON/canon_6.0_reference.md`).
