This directory contains append-only lineage and ceremony logs.

## Naming Convention

`LOG-YYYY-MM-DD-<ceremony-id>.md`

## Log Entry Fields

- **Timestamp**
- **Ceremony ID**
- **Tension reference**
- **Operators applied**
- **Artifacts changed**
- **Badges issued**
- **Witnesses**

Logs are append-only. No log entry may be modified after it is sealed.
