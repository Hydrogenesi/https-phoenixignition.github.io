This directory contains governance artifacts for the CVN·A_TUA_M system.

## Governance Layers

| Layer | Mechanism |
|---|---|
| Tension law | TUA tension cards define and authorize change |
| Operator logic | Phoenix operator selection maps tension to action |
| Scaling | Canon 6.0 prevents scope creep across scale bands |
| Ceremony | Quantumonix ceremonies make change visible and auditable |

## Principles

- Change is trauma-informed: operators are selected with awareness of impact.
- Change is reversible: lineage enables rollback.
- Change is auditable: logs and badges create a complete record.
- Change is consensual: ceremonies require witnesses.
